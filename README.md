# Codsoft Java Projects - Complete

This repository contains a **complete collection of Java projects** completed during my **Codsoft internship** as part of the JPO tasks. These projects were developed to demonstrate programming skills, problem-solving, and Java fundamentals.

## Projects Included
1. **Number Game** – A number guessing game where the user tries to guess the number chosen by the computer.  
2. **Student Grade Calculator** – Calculates students’ grades based on their marks.  
3. **ATM Interface** – A simple ATM simulation allowing deposit, withdrawal, and balance checking.  
4. **Currency Converter** – Converts currency using predefined exchange rates.  
5. **Student Management System** – A system to manage student records, including adding, updating, and deleting students.
